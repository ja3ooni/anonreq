# دليل مستندات اللغة العربية (Arabic Documentation Guide)

تتوفر مستندات بوابة AnonReq باللغة العربية في هذا الدليل. نظرًا لطبيعة اللغة العربية التي تُكتب من اليمين إلى اليسار (RTL)، يُرجى اتباع الإرشادات التالية لعرض وتنسيق المستندات بشكل صحيح.

## إرشادات العرض باتجاه اليمين إلى اليسار (RTL Rendering Guidance)

1. **دعم اتجاه النص (Text Direction Support):**
   يجب أن تدعم أدوات عرض لغة Markdown خاصية تحديد اتجاه النص `dir="rtl"` لضمان محاذاة الفقرات والقوائم والعناوين لليمين.
   
2. **علامات التنسيق الاختيارية (Direction Markers):**
   إذا لم تتعرف أداة العرض تلقائيًا على اتجاه النص العربي، يُنصح بإضافة وسم `<!-- rtl -->` أو استخدام كتل لغة HTML المحددة بـ `<div dir="rtl">` لتغليف المحتوى العربي.

3. **التعليمات البرمجية والأوامر (Code Blocks and Commands):**
   تظل جميع كتل الأكواد البرمجية وأوامر الطرفية (CLI) وملفات التكوين (YAML/JSON) مكتوبة باللغة الإنجليزية وتُعرض من اليسار إلى اليمين (LTR). من المتوقع وجود محتوى مختلط الاتجاهات (Mixed-Direction Content) في بعض الصفحات.

4. **تأجيل التحقق الكامل من التنسيق (Verification Status):**
   تم تأجيل التحقق الكامل من اتجاهات العرض الرسومي وتخطيط الصفحات لليمين إلى اليسار (RTL layout verification) في واجهات الاستخدام إلى الإصدار الثاني من البوابة (v2.0) وفقًا لمتطلبات المشروع المؤجلة.
