# Plan 21-01: Endpoint Agent Foundation (macOS + Cross-Platform Core)

**Objective**: Create the desktop endpoint agent infrastructure — local traffic capture, AI application discovery, and local model routing foundation for macOS (primary) with a cross-platform core.

## Tasks

### 1. Endpoint Agent Base

- Create `src/anonreq/endpoint/agent.py` — `EndpointAgent` async lifecycle class
- Discovery modes: passive traffic monitoring, active process enumeration
- Heartbeat + telemetry emission to gateway
- Config loaded from `endpoint.yaml`

### 2. macOS Traffic Capture

- Create `src/anonreq/endpoint/macos/capture.py` — Packet capture via pcap/NPF
- Filter AI provider traffic by hostname/IP (reuse `hostname_matcher` from discovery)
- Emit `ai_traffic_observed` events to gateway via WebSocket

### 3. AI Application Discovery

- Create `src/anonreq/endpoint/discovery.py` — Process scanner for AI apps
- Known apps: Cursor, Claude Desktop, ChatGPT Desktop, VS Code (Copilot extension identifier)
- Emit `ai_app_discovered` events with process name, bundle ID, version

### 4. Endpoint Configuration API

- `GET /v1/endpoint/status` — Agent health, uptime, discovered apps
- `GET /v1/endpoint/apps` — List discovered AI applications
- `GET /v1/endpoint/traffic` — Recent AI traffic summary

### 5. Installer Packaging

- `scripts/package-endpoint.sh` — macOS .app bundle creation
- Plist for launch-on-startup

### 6. Audit

- `endpoint_agent_started`, `endpoint_agent_stopped`, `ai_app_discovered`, `ai_traffic_observed` audit events
- No raw traffic content in audit
