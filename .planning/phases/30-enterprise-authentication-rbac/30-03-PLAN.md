<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Trusted ingress -> gateway | Forwarded certificate material is trusted only when the proxy source is trusted. |
| Certificate payload -> machine principal | The forwarded cert becomes machine-auth context only after parse/verification. |

| T-30-06 | Tampering | proxy-forwarded auth data | mitigate | Validate certificate payloads and fail closed on malformed inputs. |

</threat_model>
