    1. **src/anonreq/discovery/risk_score.py:**
       - `RiskBand` enum: `LOW` (0-30), `MEDIUM` (31-60), `HIGH` (61-80), `CRITICAL` (81-100).
       - `RiskScore` dataclass:
         - `overall: float`, `band: RiskBand`.
         - `dimensions: dict[str, float]` — individual dimension scores.
         - `weights: dict[str, float]` — weights used for calculation.
       - `DimensionScore` dataclass: `name: str`, `score: float`, `weight: float`, `inputs: dict`.
       - `RiskScoreEngine` class:
         - Constructor takes `config: dict`:
           - `config["risk_score"]["dimensions"]` — dimensions with default weights.
           - `config["risk_score"]["tenant_weights"]` — per-tenant weight overrides.
         - `calculate(context: RiskContext) -> RiskScore`:
           - Context includes all inputs needed for 6 dimensions.
           - **Provider Trust** (default weight 25%):
             - Score based on: provider tier (major=10, regional=40, unknown=80) + jurisdiction (us/eu=+0, cn=+20, other=+30, unknown=+40) + certifications (soc2/iso27001=-10 each) + SLA (yes=-10).
             - Range: 0-100.
           - **Data Sensitivity** (default weight 20%):
             - Score based on weighted average of classification levels observed in traffic: Public=10, Internal=25, Confidential=50, Restricted=75, Highly Restricted=100.
             - If no traffic data: 30 (default).
           - **Shadow Usage** (default weight 20%):
             - Score: sanctioned=10, tolerated=50, unsanctioned=90, unknown=70.
           - **Approval Status** (default weight 15%):
             - Score: approved=5, pending=50, not_reviewed=80, denied=100.
           - **Model Location** (default weight 10%):
             - Score: in-region (same as tenant)=10, same continent=30, different continent=60, unknown=90.
           - **Retention Policy** (default weight 10%):
             - Score: no retention=10, 30-day=30, 90-day=50, 1-year=70, indefinite=90, unknown=100.
           - **Weighted sum**: `overall = Σ(dimension_score × weight) / Σ(weights)`.
           - **Clamp**: overall in [0, 100].
           - **Band**: `get_band(overall)`.
         - `get_band(score: float) -> RiskBand`.
         - `set_tenant_weights(tenant_id: str, weights: dict) -> None`.
         - Recalculation: `calculate()` is deterministic — same context → same risk score.

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-05-01 | Tampering | Inventory data | mitigate | DNS/Proxy data is read-only (parsed, not modified). Manual entries require admin auth. Inventory is append-only with timeline merge. |
| T-19-05-02 | Tampering | Risk score weights | mitigate | Weight configuration is admin-only via policy YAML. Validate weights sum to ~1.0 (within 0.05 tolerance). |
