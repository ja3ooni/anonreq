## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| YAML policy file → config loader | Untrusted YAML parsed at startup or reload |
| Gateway → Valkey (policy cache) | Policy state crosses from gateway to Valkey over Redis protocol |
| Tenant header → ResidencyRouter | Tenant ID from request scopes routing decisions |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01-01 | Tampering | config.py YAML loading | mitigate | `yaml.safe_load()` prevents code injection; Pydantic validation rejects malformed structure |
| T-08-01-02 | Elevation of Privilege | PolicyStore tenant isolation | mitigate | All reads scope by tenant_id from authenticated context; no cross-tenant policy access |
| T-08-01-03 | Denial of Service | UsageLimiter INCR flood | mitigate | Bounded key TTL (60s windows), fail-closed on cache error (503) |
| T-08-01-04 | Information Disclosure | ResidencyRouter region info | mitigate | Region config is metadata-only; no raw payload or token values exposed |
| T-08-01-05 | Tampering | PolicyStore cache | mitigate | Version-pinned cache with hash validation; stale cache detected and refreshed |
</threat_model>
