## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Admin API request → RBAC middleware | Untrusted request enters authorization boundary |
| RBAC → Policy Store | Authorized request reaches persistent policy state |
| Usage API → Redis counters | Query reads live counters from Valkey |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-03-01 | Elevation of Privilege | RBAC role check | mitigate | Role hierarchy enforced via FastAPI dependency; every admin route declares minimum role |
| T-08-03-02 | Tampering | PUT policy update | mitigate | Pydantic validation rejects malformed policy bodies; version bump on update |
| T-08-03-03 | Information Disclosure | Usage endpoint output | mitigate | Counters only — no raw data, tokens, or entity values returned |
| T-08-03-04 | Spoofing | Unauthenticated access | mitigate | Auth middleware runs before RBAC; missing/invalid token returns 401 |
| T-08-03-05 | Denial of Service | Expensive policy list | mitigate | Pagination limits; PolicyStore returns cached compiled policies |
</threat_model>
