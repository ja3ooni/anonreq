<objective>
Implement the data classification engine that assigns a sensitivity level to every request based on detected entity types. This plan delivers: (1) the ClassificationEngine with deterministic highest-sensitivity algorithm, (2) the 5-level classification enum (Public, Internal, Confidential, Restricted, Highly Restricted), (3) the default entity-to-classification mapping table covering all Phase 2 entity types, and (4) YAML override support via Phase 8 policy YAML.

Purpose: Every request must be classified by sensitivity before PDP #2 policy evaluation. The deterministic max algorithm ensures auditable, explainable classification for regulated enterprises — no AI, no confidence blending.

Output: `ClassificationEngine.classify([EMAIL, PHONE])` returns `ClassificationResult(highest=CONFIDENTIAL, labels=[...])`. Entity mapping configurable per tenant.
</objective>

       ```python
       class ClassificationEngine:
           def __init__(self, entity_map: dict[str, ClassificationLevel] | None = None):
               """Initialize with optional override map (loaded from tenant policy YAML).
               Falls back to ENTITY_CLASSIFICATION_MAP defaults."""
               self._entity_map = entity_map or dict(ENTITY_CLASSIFICATION_MAP)

           async def classify(self, entity_types: list[str]) -> ClassificationResult:
               """Classify based on detected entity types using deterministic max algorithm.
               1. For each entity type, look up classification level
               2. Unknown types → INTERNAL (log warning)
               3. highest = max(levels)
               4. If empty input → INTERNAL (undetected defaults to Internal per CLASS-02)
               5. Return ClassificationResult with highest, all labels, detected levels"""
               if not entity_types:
                   return ClassificationResult(
                       highest=ClassificationLevel.INTERNAL,
                       labels=[],
                       detected_levels=[],
                   )
               labels: list[str] = []
               levels: list[ClassificationLevel] = []
               for et in entity_types:
                   level = self._entity_map.get(et.upper())
                   if level is None:
                       level = ClassificationLevel.INTERNAL
                   labels.append(et)
                   levels.append(level)
               highest = max(levels)
               return ClassificationResult(
                   highest=highest,
                   labels=labels,
                   detected_levels=levels,
               )

           async def classify_with_client_override(
               self, entity_types: list[str],
               client_level: ClassificationLevel | None = None,
           ) -> ClassificationResult:
               """Classify and apply client-asserted override (increase only).
               Higher of detected vs client wins. Override logged."""
               result = await self.classify(entity_types)
               if client_level is not None:
                   if client_level > result.highest:
                       result.highest = client_level
                       result.client_override = True
                       result.client_asserted_level = client_level
               return result

           def update_entity_map(self, overrides: dict[str, ClassificationLevel]) -> None:
               """Merge tenant-specific overrides into entity map."""
               self._entity_map.update(overrides)
       ```
       - Algorithm: `highest = max(entity_mapping[e] for e in detected_entities)`
       - Purely deterministic: same entity types → same classification every time
       - No AI, no scoring, no confidence blending per D-005
       - Unknown entity types default to INTERNAL with a `logger.warning`
