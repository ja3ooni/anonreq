<objective>
Extract and anonymize tool call arguments from all three supported provider formats: OpenAI tool_calls array, Anthropic tool_use content blocks, and MCP tool call/result payloads. This plan delivers: (1) OpenAI tool_calls argument extraction — parse arguments JSON, anonymize string values, reassemble; (2) Anthropic tool_use extraction — locate tool_use blocks in content array, extract and anonymize input; (3) MCP protocol extraction — tool call and result payload handling; (4) Tool result (tool role) anonymization on the response path.

Purpose: Tool call arguments carry structured data that frequently contains PII (customer names, account numbers, addresses). Without this capability, PII in tool calls would bypass the anonymization pipeline entirely.
</objective>
