    3. For remaining violations (typically ARG, SIM, B categories which are unsafe to auto-fix):
       - Fix each category manually by editing the affected files.
       - ARG violations: add leading underscore to unused function arguments or remove them.
       - SIM violations: simplify expressions where safe (e.g., use `if x` instead of `if x is True`).

    4. Run `ruff check src/ tests/` and continue fixing until exit code 0 with no violations.
    5. Run `ruff format --check src/ tests/` to ensure formatting is clean (do not reformat — just verify).

    Per D-07: Auto-fix safe violations automatically. For unsafe ones (ARG, SIM, B), fix manually per category.
