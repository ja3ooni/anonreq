<objective>
Implement MCP protocol parsing and agent/tool-call governance — the capability to inspect, validate, and sanitize agent tool calls and tool results. This wave delivers: (1) MCP message frame detection and parsing with schema mapping, (2) tool call argument inspection via AI Firewall scan and schema enforcement, (3) tool result anonymization through the Detection Engine with JSON key preservation, and (4) error redaction for stack traces, internal IPs, and environment variables.

Purpose: AI agents are the fastest-growing AI attack surface. Without this wave, agent tool calls carry sensitive data to external tools and tool results bring sensitive data back to the LLM — both unchecked.

Output: MCP message parsing and tool call/result governance with injection detection, tokenization, error redaction, and full audit coverage.
</objective>

         - `__init__(self, firewall, schema_registry: dict, config: ToolGovernanceConfig)`
         - `async def inspect_call(self, tool_call: ToolCall) -> InspectionResult`:
           1. Look up tool in `per_tool_policies` — if `block`, return block immediately
           2. If `schema_registry` has schema for tool_name: validate arguments against schema
              - Missing required fields → block

         - `ENV_VAR_RE = re.compile(r'\$[A-Z_][A-Z0-9_]+|%[A-Z_][A-Z0-9_]+%')`

         - `__init__(self, detection_engine, tokenization_engine, config: ToolGovernanceConfig)`
         - `async def sanitize_result(self, result: ToolResult) -> ToolResult`:
           1. Run Detection Engine on all string values in result.content
           2. Tokenize detected entities with TokenizationEngine
           3. Preserve all JSON keys (never modify field names per D-019)
           4. Run error redaction on all string values
           5. Return sanitized ToolResult with original structure
         - `def _redact_errors(self, value: str) -> str`:
           - Replace ERROR_PATTERNS matches with `[REDACTED_ERROR]`
         - `def _traverse_and_sanitize(self, obj: Any, path: str = "") -> Any`:
           - Recursively traverse JSON, sanitize string values, preserve non-string types
           - Never modify dict keys — only string values
