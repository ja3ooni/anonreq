<objective>
Implement the audio stream sanitization layer — the core privacy enforcement for the voice pipeline. This wave delivers: (1) overlapping sliding-window detection that maps entity positions from STT transcript text to millisecond-range audio frames, (2) two sanitization output paths (text tokenization and audio muting/beeping), (3) outbound resynchronization of sanitized frames into the stream.

Purpose: Audio sanitization is the differentiator for voice AI protection. Without this wave, voice streams can be transcribed but PII in spoken content cannot be redacted from the audio itself.

Output: A working VoicePipeline that ingests audio, transcribes, detects sensitive entities, and outputs sanitized audio/text with <150ms P99 added latency.
</objective>

         - `def char_offset_to_ms(self, char_offset: int, text_start_ms: int, text_duration_ms: int, text_length: int) -> int`:

              c. Map entity positions to audio timeline via TimelineMapper
              d. Return list of timeline-localized entity dicts
              e. Prune overlapped portion for next window

    Implement the audio sanitizer and voice pipeline orchestrator per D-015, D-016.

    1. **src/anonreq/voice/sanitizer.py** — Per D-015 (audio path):
       - `class AudioSanitizer`:
         - `__init__(self, config: VoiceConfig)`
         - `async def mute_frame(self, audio_data: bytes, sample_rate: int, start_ms: int, end_ms: int) -> bytes`:
           - Replace audio samples in [start_ms, end_ms] with silence (zeros)
           - Calculate byte offset: `start_ms * sample_rate * bytes_per_sample / 1000`
         - `async def beep_frame(self, audio_data: bytes, sample_rate: int, start_ms: int, end_ms: int, frequency: int = 1000, amplitude: float = 0.5) -> bytes`:
           - Generate sine wave at frequency, overlay on entity range
           - `t = np.arange(0, duration_s, 1/sample_rate)`
           - `beep = amplitude * np.sin(2 * np.pi * frequency * t)`
         - `async def sanitize(self, audio_data: bytes, entities: list[dict], method: str = "mute", sample_rate: int = 16000) -> bytes`:
           - Apply muting or beeping for each entity's frame range
           - Handle overlapping ranges (single combined range)
           - Return sanitized audio data
         - `class TextSanitizer`:
           - `async def sanitize_text(self, text: str, entities: list[dict]) -> str`:
             - Replace entity spans with [TYPE_N] tokens
             - Reuse Phase 2 TokenizationEngine patterns

    2. **src/anonreq/voice/pipeline.py**:
       - `class VoicePipeline`:
         - `__init__(self, stt_engine: STTEngine, detector: SlidingWindowDetector, audio_sanitizer: AudioSanitizer, text_sanitizer: TextSanitizer, config: VoiceConfig)`
         - `async def process_stream(self, connector: BaseConnector, output_format: str = "audio") -> AsyncIterator[bytes | str]`:
           - Main orchestrator loop:
             1. Receive AudioChunk from connector
             2. Transcribe chunk via STTEngine (if output is audio path)
             3. Run SlidingWindowDetector on transcript
             4. If entities found: sanitize audio (mute/beep) or text (tokenize)
             5. Yield sanitized frame or text
             6. Track latency per chunk → emit voice_latency_exceeded if >150ms
         - `async def start(self) -> None`, `async def stop(self) -> None`
         - Prometheus: `anonreq_voice_latency_ms` histogram, `anonreq_voice_streams_active` gauge
         - Audit events: voice_stream_started, voice_stream_ended, voice_entity_detected, voice_audio_sanitized, voice_latency_exceeded
