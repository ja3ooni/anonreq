           session_id: str

       Content stripping logic:
       ```python
       STRIP_FIELDS = {"content", "prompt", "response", "raw_text", "message", "text"}

       def _strip_content_fields(self, raw: RawSecurityEvent) -> dict:
           stripped: dict[str, Any] = {}
           found_content = False
           for key, value in raw.content.items():
               if key.lower() in STRIP_FIELDS:
                   found_content = True
                   continue
               stripped[key] = value
           self._content_stripped = found_content
           return stripped
       ```

| Detection Engine → Event Bus | Raw security events cross from detection engines into SOC pipeline |
| Event Normalizer → Sink Router | Normalized events forwarded to per-sink formatters |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-20-01-02 | Tampering | mitre-mapping.yaml | mitigate | yaml.safe_load() prevents code execution. Validation catch missing/invalid fields at startup (D-013). |
