    Create curl scripts (examples/curl/) following the same trap-based pass/fail pattern
    as quickstart scripts (set -euo pipefail, PASSED trap, timing, exit 0/1). Each script
    reads ANONREQ_API_KEY from env (default test-key-0123456789abcdef). The gateway is
    expected to be running at localhost:8000.

    curl/basic-anonymization.sh:
    - POST to /v1/chat/completions with message containing email and phone PII.
    - Assert [EMAIL_1] and [PHONE_1] in response content using python3 -c json parsing.
    - Print tokenized response on success.

    curl/streaming.sh:
    - POST with stream: true, capture SSE chunks.
    - Verify response contains data: prefixed lines with [DONE] termination.
    - Pass if no errors, fail on curl error or missing [DONE].

    curl/gdpr-preset.sh:
    - POST to /v1/chat/completions with X-AnonReq-Compliance-Preset: gdpr header.
    - Send message containing German PII (Steuer-ID, phone).
    - Assert tokens present in response.

    curl/locale-de.sh:
    - POST with X-AnonReq-Locale: de-DE header.
    - Send message with German PII (name, address, Steuer-ID).
    - Assert [NAME_1], [STREUR_ID_1] or equivalent tokens present.

    Create Python examples (examples/python/) as standalone projects. Each has:
    - requirements.txt listing `requests` only
    - main.py as executable script (if __name__ == "__main__")
    - README.md with usage: `pip install -r requirements.txt && python main.py`

    python/basic-anonymization/main.py:
    - Import requests, json, sys, os.
    - Read ANONREQ_API_KEY from env, default test-key-0123456789abcdef.
    - Set API_URL default http://localhost:8000/v1/chat/completions.
    - Build payload with model "gpt-4o" and message containing email + phone PII.
    - POST with Authorization header, print response JSON.
    - Exit non-zero if status != 200.
    - Show "[EMAIL_1]" and "[PHONE_1]" in output on success.

    python/streaming/main.py:
    - Similar setup but with stream=True.
    - Iterate over response.iter_lines().
    - Print each line prefixed with "data: ".
    - Check for "[DONE]" termination event.
    - Exit non-zero if streaming fails or [DONE] missing.

    python/gdpr-preset/main.py:
    - Add X-AnonReq-Compliance-Preset: gdpr header.
    - Send message with German PII.
    - Assert tokens present in response.

    python/locale-de/main.py:
    - Add X-AnonReq-Locale: de-DE header.
    - Send German-language message with name, address.
    - Assert tokens present in response.

    Per D-196: standalone runnable projects, not markdown-only snippets.
    Per D-197: 4 examples per language covering Basic, Streaming, GDPR Preset, Locale DE.
    Per D-198: production-grade, CI-tested as acceptance tests.
    Per D-199: gateway repo is canonical source — examples never drift from main codebase.
    Per D-200: docs reference examples; examples are the single source of truth.
