<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Log event -> serializer | Sensitive substrings must be redacted before output leaves the logger. |
| Rotation buffer -> active stream | A running SSE stream may keep using an older key snapshot until it ends. |
| New stream -> latest snapshot | New work must not inherit stale keys unless it explicitly started before rotation. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|----------|----------|-----------|-------------|-----------------|
| T-29-05 | Information Disclosure | log serialization | mitigate | Redact secret substrings before the JSON renderer executes. |
| T-29-06 | Tampering | key rotation boundary | mitigate | Keep previous keys read-only and versioned so active streams cannot observe partial updates. |

</threat_model>
