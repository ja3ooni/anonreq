       - `FINANCIAL_ENTITY_TYPES = ["IBAN", "PAYMENT_REF", "CUSTOMER_ID", "AML_CASE_REF"]`
       - `class ContextBooster`:
         - `__init__(self, config_path: str = "config/financial_crime_words.yaml")`: loads config
         - `def load_config(self) -> dict`: reads YAML, returns config dict
         - `def is_within_proximity(self, entity_start: int, entity_end: int, word_positions: list[tuple[int, int]], proximity: int) -> bool`:
           - Checks if any high-risk word position is within `proximity` chars of the entity span
         - `def find_high_risk_word_positions(self, text: str, high_risk_words: list[str]) -> list[tuple[int, int]]`:
           - Returns (start, end) positions for all high-risk word matches in text
         - `def apply_boost(self, entity: DetectionResult, text: str, word_positions: list[tuple[int, int]]) -> DetectionResult`:
           - If entity.entity_type in FINANCIAL_ENTITY_TYPES AND `is_within_proximity(entity.start, entity.end, word_positions, self.proximity)`:
             - `new_score = min(entity.score + self.boost_amount, 1.0)`
           - Returns updated DetectionResult with modified score

         ```python
             word_positions = booster.find_high_risk_word_positions(original_text, high_risk_words)
             boosted_detections = [booster.apply_boost(d, original_text, word_positions) for d in detections]
             # ... deduplication ...
         ```
