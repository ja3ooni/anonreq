       ```python
       @dataclass
       class ProcessingContext:
           request_id: str
           tenant_id: str = "default"
           context_id: str | None = None        # UUIDv7 hex for mapping key
           original_request: dict | None = None  # Parsed ChatRequest as dict
           text_nodes: list[dict] | None = None  # TextNode list from extraction
           classification_result: dict | None = None  # {"action": "...", "matched_rule_ids": [...]}
           detections: list[dict] | None = None       # [{"entity_type": "...", "start": N, "end": N, "score": F}]
           token_mappings: dict[str, str] | None = None  # {"[EMAIL_0]": "user@example.com"}
           transformed_request: dict | None = None       # Sanitized request body
           provider_response: dict | None = None          # Raw LLM response
           restored_response: dict | None = None          # Response with tokens restored
           audit_metadata: dict = field(default_factory=dict)
           errors: list[Exception] = field(default_factory=list)

           def has_errors(self) -> bool:
               return len(self.errors) > 0

           def fail_secure(self, error: Exception) -> None:
               """Record error and abort pipeline. Per D-49 any stage failure aborts."""
               self.errors.append(error)
       ```
       Uses `from dataclasses import dataclass, field`. Fields default to None until populated by the relevant stage.

    3. **src/anonreq/models/chat.py** — Pydantic models matching OpenAI chat completions schema per PIPE-01 (from RESEARCH Example 3):
       - `ChatMessage(BaseModel)`: role (system/user/assistant/tool/function), content (str | list | None), name, tool_calls, tool_call_id
       - `ChatRequest(BaseModel)`: model, messages (list[ChatMessage]), stream (bool=False), temperature, top_p, n, stop, max_tokens, presence_penalty, frequency_penalty, logit_bias, user, tools, tool_choice, response_format, seed
       - `ChatCompletionChoice(BaseModel)`: index, message (dict), finish_reason
       - `ChatCompletionResponse(BaseModel)`: id, object="chat.completion", created, model, choices (list[ChatCompletionChoice]), usage (dict | None)
       - Add `model_config = {"extra": "ignore"}` to all chat models for forward compatibility with API changes.
       - Do NOT include `functions` or legacy fields — they are deprecated in the OpenAI API.

    4. **src/anonreq/models/classification.py** — per D-22 through D-28:
       - `ClassificationRule` dataclass: id, enabled, version, name, action (BLOCK/ROUTE_LOCAL/ANONYMIZE/PASS), metadata dict, roles list, regex_patterns list (raw strings), keywords list
       - `ClassResult` dataclass: action str, matched_rule_ids list[str], matched_rule_versions list[int]
       - `ClassificationAction` str enum or Literal type: `BLOCK`, `ROUTE_LOCAL`, `ANONYMIZE`, `PASS`

    5. **src/anonreq/models/detection.py** — per D-29 and D-33:
       - `TextNode` dataclass: path (str), role (str), value (str) — represents one extracted text fragment
       - `DetectionResult` dataclass: entity_type (str), start (int), end (int), score (float), source (Literal["regex", "ner"])

       - Token pattern regex: `r'\[[A-Z][A-Z_]{0,19}_\d+\]'` as module-level constant for reuse by restorer

    All model files import from stdlib only (dataclasses, typing, uuid) plus pydantic.BaseModel for chat.py.

    2. **src/anonreq/cache/manager.py** — per RESEARCH Example 4:
       - `CacheManager` class with constructor taking `redis_url: str, ttl: int = 300`:
         - Creates connection pool via `redis.asyncio.from_url(redis_url, decode_responses=True, health_check_interval=5, socket_connect_timeout=3)`
         - Stores `self._ttl` for EXPIRE
       - `def _key(self, tenant_id: str, session_id: str) -> str`:
         - Returns `f"anonreq:{tenant_id}:{session_id}"` per D-13
       - `async def store_mapping(self, tenant_id: str, session_id: str, mapping: dict[str, str]) -> None`:
         - Atomic HSET + EXPIRE via pipeline(transaction=True) per D-14
         - `pipe.hset(key, mapping=mapping)` then `pipe.expire(key, self._ttl)` then `pipe.execute()`
       - `async def get_mapping(self, tenant_id: str, session_id: str) -> dict[str, str]`:
         - `hgetall(key)` returns all token→value pairs per D-15
       - `async def delete_mapping(self, tenant_id: str, session_id: str) -> None`:
         - `await self._redis.delete(key)` per CACH-04
       - `async def close(self) -> None`:
         - `await self._redis.aclose()` for clean shutdown
       - Import `redis.asyncio as redis` — the redis-py 8.0+ async client

    3. **src/anonreq/cache/health.py** — per CACH-06:
       - `async def check_cache_health(manager: CacheManager) -> dict`:
         - Ping Valkey: `await manager._redis.ping()`

         - Return `{"reachable": bool, "persistence_disabled": bool, "healthy": bool}`
         - On exception, return dict with `healthy: False` and error info
