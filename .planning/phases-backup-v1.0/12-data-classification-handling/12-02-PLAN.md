<objective>
Integrate the classification engine into the request pipeline with client-asserted classification headers and PDP #2 integration. This plan delivers: (1) X-AnonReq-Classification header parsing with increase-only validation, (2) classification stamping on RequestContext after the detection stage, (3) pipeline stage for classification, and (4) PDP #2 integration for per-level handling policy evaluation.

Purpose: Classification must run after entity detection and before PDP #2 so that policy decisions are sensitivity-aware. Client-asserted classification allows enterprise workflows (M&A docs, trade secrets) where detection alone is insufficient.

Output: Pipeline executes classification after detection. Requests with Highly Restricted classification are blocked at PDP #2 (HTTP 451). Client override logged.
</objective>

       ```python
       CLASSIFICATION_HEADER = "X-AnonReq-Classification"
       DEBUG_HEADER = "X-AnonReq-Debug"

       def parse_classification_header(request: Request) -> ClassificationLevel | None:
           """Parse X-AnonReq-Classification header.
           Returns None if header is absent.
           Raises HTTPException(400) if value is invalid."""
           value = request.headers.get(CLASSIFICATION_HEADER)
           if value is None:
               return None
           try:
               return ClassificationLevel[value.upper().replace(" ", "_")]
           except KeyError:
               raise HTTPException(
                   status_code=400,
                   detail=f"Invalid classification level: '{value}'. "
                          f"Valid: Public, Internal, Confidential, Restricted, Highly Restricted",
               )

       def validate_classification_override(
           detected: ClassificationResult,
           client_level: ClassificationLevel | None,
       ) -> ClassificationResult:
           """Apply increase-only validation.
           Higher of detected vs client wins. Never decrease.
           Returns updated ClassificationResult with override flag."""
           if client_level is None:
               return detected
           if client_level < detected.highest:
               # Client attempted to decrease — log and ignore
               logger.warning(
                   "client_classification_decrease_rejected",
                   detected=detected.highest.name,
                   client_attempted=client_level.name,
               )
               return detected
           if client_level > detected.highest:
               # Client increased — this is allowed
               detected.highest = client_level
               detected.client_override = True
               detected.client_asserted_level = client_level
               logger.info(
                   "client_classification_override",
                   original=detected.highest.name,
                   override=client_level.name,
               )
           return detected
       ```

       class PipelineService:
           def __init__(self, ..., classification_engine: ClassificationEngine):
               self._classification_engine = classification_engine
               # Add classification to stage sequence: detection → classification → anonymization → ...

           async def _run_classification(self, ctx: ProcessingContext) -> None:
               """Classification stage: runs after detection, before anonymization.
               1. Extract entity types from ctx.detections
               2. Call ClassificationEngine.classify(entity_types)
               3. Apply client header override if present
               4. Stamp result on ctx.classification_result
               5. Emit classification_computed audit event"""
               entity_types = list(set(d["entity_type"] for d in (ctx.detections or [])))
               result = await self._classification_engine.classify(entity_types)
               client_level = ctx.client_classification_header
               result = validate_classification_override(result, client_level)
               ctx.classification_result = result

               if result.client_override:
                   logger.info("classification_client_override",
                               original=result.highest.name,
                               client_level=client_level.name)

           async def run(self, ctx: ProcessingContext) -> ProcessingContext:
               """Execute pipeline stages in order."""
               # ... existing stages ...
               await self._run_classification(ctx)  # After detection
               # Check PDP #1 and #2 before anonymization
               pdp2_decision = await self._pdp2.evaluate(ctx)
               if pdp2_decision.action == "BLOCK":
                   ctx.fail_secure(PipelineBlockedError(pdp2_decision))

    2. **src/anonreq/services/pdp2.py** — Add classification-aware policy evaluation:
       ```python
       class PDP2Service:
           # ... existing code ...

           async def evaluate(self, ctx: ProcessingContext) -> PolicyDecision:
               """Evaluate policies against processing context.
               Includes classification-aware per-level handling."""
               classification = ctx.classification_result

               # Default per-level handling actions
               level_actions = {
                   ClassificationLevel.PUBLIC: PolicyAction.PASS,
                   ClassificationLevel.INTERNAL: PolicyAction.PASS,
                   ClassificationLevel.CONFIDENTIAL: PolicyAction.ANONYMIZE,
                   ClassificationLevel.RESTRICTED: PolicyAction.ANONYMIZE_AND_AUDIT,
                   ClassificationLevel.HIGHLY_RESTRICTED: PolicyAction.BLOCK,
               }

               # Check tenant override from policy YAML
               if classification and classification.highest in self._tenant_policies.get(
                   ctx.tenant_id, {}
               ).get("classification", {}).get("handling", {}):
                   override = self._tenant_policies[ctx.tenant_id]["classification"]["handling"][
                       classification.highest.name
                   ]
                   action = PolicyAction(override["action"])
               elif classification:
                   action = level_actions.get(classification.highest, PolicyAction.PASS)
               else:
                   action = PolicyAction.PASS

               if action == PolicyAction.BLOCK:
                   return PolicyDecision(
                       action="BLOCK",
                       status_code=451,
                       detail=f"Request classified as {classification.highest.display_name} "
                              f"is blocked per policy",
                       audit_event_type="classification_blocked",
                   )

               return PolicyDecision(
                   action=action.value,
                   status_code=200,
                   audit_event_type="classification_applied",
               )
       ```
       - Classification executed after detection, before PDP #2
       - PDP #2 uses classification to determine per-level action
       - Per-level actions: Public→PASS, Internal→PASS, Confidential→ANONYMIZE, Restricted→ANONYMIZE+AUDIT, Highly Restricted→BLOCK
