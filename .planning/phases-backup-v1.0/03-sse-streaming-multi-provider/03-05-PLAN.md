## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client -> Gateway chat route | Untrusted OpenAI-compatible request body sets `stream:true` and model alias. |
| Gateway -> Provider adapter | Sanitized prompt leaves the gateway through provider-specific adapter streaming. |
| Provider stream -> Gateway SSE | Untrusted upstream stream events are normalized and restored before returning to client. |
| Gateway -> Cache cleanup | Sensitive token mapping is deleted after stream termination or disconnect. |

| T-03-05-02 | Tampering | Model alias resolution | mitigate | Resolve aliases before adapter dispatch; unknown aliases return HTTP 400 and never forward upstream. |
| T-03-05-03 | Information Disclosure | Provider error handling | mitigate | Generic SSE error frames only; no API keys, upstream URLs, raw provider response bodies, or raw request content. |
