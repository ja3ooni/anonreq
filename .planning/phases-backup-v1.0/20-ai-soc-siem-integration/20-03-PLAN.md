       - `OAUTH2_SCOPE = "https://monitor.azure.com//.default"`
       - `TOKEN_GRANT_TYPE = "client_credentials"`
       - `class SentinelDCRSink`:
         - `__init__(self, name: str, tenant_id: str, client_id: str, client_secret: str, dcr_endpoint: str, dcr_immutable_id: str, stream_name: str, tls_verify: bool = True)`

           - `stream_name`: Custom table stream name
         - `async def _acquire_token(self) -> str`:
           - POST to `https://login.microsoftonline.com/{tenant_id}/oauth2/v2.0/token`
           - Body: `client_id={client_id}&client_secret={client_secret}&scope={scope}&grant_type={grant_type}`
           - Cache token with expiry metadata; refresh within 5min of expiry
         - `async def format_event(self, event: NormalizedEvent) -> dict`:
           - Return event dict matching DCR stream schema (custom table columns)
         - `async def send_event(self, event: NormalizedEvent) -> bool`:
           - Acquire token, POST to DCR endpoint with Bearer auth
           - URL: `{dcr_endpoint}/dataCollectionRules/{dcr_immutable_id}/streams/{stream_name}?api-version=2023-01-01`
         - `async def health_check(self) -> SinkStatus`:
           - Try token acquisition; if fails, return unreachable
