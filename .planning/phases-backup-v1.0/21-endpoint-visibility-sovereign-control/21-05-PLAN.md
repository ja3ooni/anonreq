<objective>
Implement the AI Firewall — the inline security gate that blocks prompt injection, jailbreak attempts, system prompt override, and adversarial manipulation before any downstream processing occurs. This wave delivers: (1) local ONNX-optimized classifier with jailbreak pattern database, (2) system prompt override detection, (3) semantic injection scoring via vector embedding distance, (4) MITRE ATLAS mapping for all firewall events, and (5) HTTP 403 response for blocked requests, all within a 20ms P99 latency budget.

Purpose: The AI Firewall is the first line of defense for all inbound AI traffic. Blocked requests never reach the detection pipeline, saving compute and preventing model spend on malicious traffic.

Output: FirewallPipeline that runs inline before PDP #1 with full classification, scoring, and MITRE ATLAS audit — all under 20ms P99.
</objective>
